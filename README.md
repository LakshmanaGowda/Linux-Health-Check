# Linux Health Check Script

A Bash-based Linux Health Check utility developed as part of my DevOps learning roadmap.

## Features

- System Information
- System Uptime
- Memory Usage Report
- Memory Health Analysis
- Disk Usage Report
- Disk Health Analysis
- ANSI Color Output
- Timestamped Logging

## Technologies Used

- Bash Shell Scripting
- Linux (Ubuntu WSL2)
- awk
- tr
- tee
